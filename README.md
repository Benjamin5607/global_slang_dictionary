# Multilingual Slang & Slur Database

This project collects and analyzes slang and slur terms across 14 languages
to support research in online safety, content moderation, and sociopolitical analysis.

## Features
- Multilingual slang crawling
- English-based semantic analysis
- Severity and target group classification
- Google Sheets integration
- LLM-efficient pipeline

## Languages
EN, KO, JA, TH, VI, ID, MS, ES, PT, TL, DE, FR, RU, IT
